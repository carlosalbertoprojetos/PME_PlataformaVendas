# 05 - Guia do vendedor

## Objetivo

Ensinar o vendedor a usar o sistema para vender melhor, acompanhar clientes e nao esquecer follow-ups.

## Publico

Vendedores internos, representantes, pre-vendas e atendentes comerciais.

## Quando utilizar

Use todos os dias para organizar contatos, clientes, oportunidades, WhatsApp e proximas acoes.

## Passo a passo

1. Entre no sistema.
2. Abra sua agenda ou dashboard.
3. Veja acoes de hoje e atrasadas.
4. Abra o cliente ou oportunidade.
5. Consulte historico, pedidos e recomendacoes.
6. Envie WhatsApp quando houver telefone valido.
7. Registre o que aconteceu.
8. Crie a proxima acao.
9. Atualize a oportunidade quando necessario.
10. Volte para a agenda ate concluir o dia.

## Exemplos

- Cliente sem recompra: enviar mensagem de retomada.
- Oportunidade parada: ligar e marcar retorno.
- Pedido pendente: confirmar se o cliente deseja fechar.

## Boas praticas

- Nunca termine atendimento sem proxima acao.
- Use mensagens prontas, mas personalize quando fizer sentido.
- Consulte historico antes de falar com o cliente.
- Mantenha telefone do cliente correto.
- Registre observacoes curtas e claras.

## Erros comuns

- Fazer contato sem olhar historico.
- Enviar WhatsApp para telefone errado.
- Criar oportunidade duplicada.
- Deixar acao sem data.
- Esperar o sistema vender sozinho.

## Checklist

- [ ] Vi minha agenda.
- [ ] Resolvi acoes atrasadas.
- [ ] Abri clientes prioritarios.
- [ ] Usei recomendacoes.
- [ ] Enviei WhatsApp quando adequado.
- [ ] Criei proximas acoes.
