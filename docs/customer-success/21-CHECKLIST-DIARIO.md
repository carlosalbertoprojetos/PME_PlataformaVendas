# 21 - Checklist diario

## Objetivo

Orientar a rotina comercial de cada dia.

## Publico

Vendedores, gestores e administradores.

## Quando utilizar

Use no inicio e no fim do expediente.

## Passo a passo

1. Abra o dashboard.
2. Veja recomendacoes.
3. Veja proximas acoes atrasadas.
4. Veja acoes de hoje.
5. Abra clientes prioritarios.
6. Envie WhatsApp quando fizer sentido.
7. Registre historico.
8. Crie nova proxima acao.
9. Confira pedidos pendentes.
10. Feche o dia revisando o que ficou pendente.

## Exemplos

- Pela manha: priorizar atrasadas.
- Depois do almoco: executar follow-ups.
- Fim do dia: criar proximas acoes para amanha.

## Boas praticas

- Resolva atrasadas primeiro.
- Nao deixe cliente sem proxima acao.
- Use recomendacoes para escolher prioridade.
- Nao espere a memoria lembrar.

## Erros comuns

- Ignorar agenda.
- Fazer contato sem registrar.
- Abrir muitas telas e nao concluir nada.
- Deixar acao sem data.

## Checklist

- [ ] Dashboard aberto.
- [ ] Recomendacoes lidas.
- [ ] Atrasadas revisadas.
- [ ] Acoes de hoje executadas.
- [ ] WhatsApp enviado quando adequado.
- [ ] Historico atualizado.
- [ ] Proximas acoes criadas.
