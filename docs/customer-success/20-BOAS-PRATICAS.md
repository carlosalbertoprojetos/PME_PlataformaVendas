# 20 - Boas praticas

## Objetivo

Definir rotinas simples para manter a operacao comercial organizada.

## Publico

Toda a equipe comercial, gestores e administradores.

## Quando utilizar

Use como regra de uso diario, semanal e mensal.

## Passo a passo

1. Todos os dias, veja dashboard e agenda.
2. Toda semana, revise oportunidades e clientes sem retorno.
3. Todo mes, revise plano, limites, produtos e indicadores.
4. Registre contato importante.
5. Crie proxima acao sempre.
6. Use WhatsApp com contexto.

## Exemplos

- Rotina diaria: ver atrasadas, abrir recomendacoes, contactar clientes.
- Rotina semanal: limpar oportunidades sem acao.
- Rotina mensal: revisar produtos e plano.

## Boas praticas

- Menos dados, mas dados corretos.
- Uma venda em andamento deve ter oportunidade.
- Uma conversa importante deve ter historico.
- Uma promessa deve virar proxima acao.
- Um cliente parado deve receber follow-up.

## Erros comuns

- Cadastrar muito e acompanhar pouco.
- Deixar oportunidade sem dono.
- Ignorar dados duplicados.
- Nao revisar limites.
- Achar que automacao corrige processo ruim.

## Checklist

- [ ] Rotina diaria definida.
- [ ] Rotina semanal definida.
- [ ] Rotina mensal definida.
- [ ] Equipe treinada.
- [ ] Gestor acompanha indicadores.
