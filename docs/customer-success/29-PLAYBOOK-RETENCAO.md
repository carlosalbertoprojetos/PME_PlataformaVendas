# 29 - Playbook retencao

## Objetivo

Reduzir churn ajudando clientes a criar rotina e perceber valor continuamente.

## Publico

Customer Success, suporte, gestores e comercial.

## Quando utilizar

Use quando uso cair, cliente reclamar, onboarding travar ou renovacao se aproximar.

## Passo a passo

1. Veja se o cliente acessa.
2. Veja se cadastrou produtos e clientes.
3. Veja se usa proximas acoes.
4. Veja se compartilha catalogo.
5. Veja se acompanha dashboard.
6. Pergunte qual resultado espera.
7. Monte plano de recuperacao simples.
8. Acompanhe por duas semanas.

## Exemplos

- Cliente nao acessa: agendar reativacao de 30 minutos.
- Cliente usa, mas nao ve valor: focar em uma rotina comercial.
- Cliente bate limite: avaliar upgrade ou limpeza.

## Boas praticas

- Agir antes do cancelamento.
- Usar dados de uso.
- Focar em uma dor real.
- Criar plano pequeno e pratico.

## Erros comuns

- Esperar cliente pedir ajuda.
- Mandar tutorial sem entender problema.
- Vender upgrade para cliente sem uso.
- Ignorar gestor.

## Checklist

- [ ] Uso revisado.
- [ ] Dor do cliente entendida.
- [ ] Plano de recuperacao criado.
- [ ] Reuniao marcada.
- [ ] Resultado acompanhado.
