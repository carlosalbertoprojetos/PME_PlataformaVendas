# 34 - Empresa demo

## Objetivo

Explicar como usar a empresa de demonstracao para treinamento, validacao visual e demos internas.

## Publico

Equipe interna, Customer Success, suporte, comercial e implantacao.

## Quando utilizar

Use em ambiente local ou staging, nunca como base de cliente real em producao.

## Passo a passo

1. Execute o comando de usuario demo quando permitido.
2. Entre com o usuario demo.
3. Confira a empresa `Empresa Demonstração`.
4. Use dados ficticios ou anonimizados.
5. Teste dashboard, CRM, Workspace, WhatsApp e plano.
6. Nao misture dados reais de clientes.

## Exemplos

- Usuario: `carlosalberto`.
- Senha: `@Testando123`.
- Empresa: `Empresa Demonstração`.
- Uso: demonstrar dashboard executivo.

## Boas praticas

- Usar somente local/staging.
- Limpar dados quando ficarem confusos.
- Avisar que demo nao e producao.
- Nao cadastrar clientes reais sem autorizacao.

## Erros comuns

- Usar demo para piloto real.
- Enviar WhatsApp para telefone real por engano.
- Mostrar dados sensiveis em treinamento.
- Executar comando demo em producao.

## Checklist

- [ ] Ambiente nao e producao.
- [ ] Usuario demo criado.
- [ ] Empresa demo conferida.
- [ ] Dados reais nao foram usados indevidamente.
- [ ] Demo encerrada com seguranca.
