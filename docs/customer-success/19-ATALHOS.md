# 19 - Atalhos

## Objetivo

Mostrar caminhos rapidos para operacoes comuns.

## Publico

Vendedores, gestores, administradores e suporte.

## Quando utilizar

Use quando ja souber o que quer fazer e quiser chegar mais rapido.

## Passo a passo

1. Escolha a operacao.
2. Siga o caminho indicado.
3. Conclua a acao.
4. Registre proxima acao quando envolver cliente.

## Exemplos

- Ver prioridades do dia: Dashboard -> Recomendacoes -> Proximas acoes.
- Atender cliente: CRM -> Clientes -> Abrir cliente -> Workspace.
- Fazer follow-up: Workspace -> WhatsApp -> Criar proxima acao.
- Conferir plano: Dashboard -> Plano.
- Ver automacoes: Dashboard -> Automacoes.

## Boas praticas

- Use dashboard como ponto de partida.
- Use Workspace para atendimento.
- Use Agenda para organizar o dia.
- Use Catalogo para vender por WhatsApp.

## Erros comuns

- Procurar pedido fora do Workspace.
- Abrir cliente sem olhar recomendacoes.
- Enviar catalogo sem follow-up.
- Usar menu Plano sem ser admin.

## Checklist

- [ ] Sei abrir dashboard.
- [ ] Sei abrir clientes.
- [ ] Sei abrir Workspace.
- [ ] Sei abrir agenda.
- [ ] Sei compartilhar catalogo.
