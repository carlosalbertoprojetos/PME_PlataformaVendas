# 11 - Guia Catalogo

## Objetivo

Explicar como usar o catalogo para apresentar produtos e iniciar conversas comerciais.

## Publico

Vendedores, gestores e administradores.

## Quando utilizar

Use quando quiser enviar produtos para um cliente, iniciar venda por WhatsApp ou divulgar itens ativos.

## Passo a passo

1. Confira se existem produtos ativos.
2. Abra Catalogo.
3. Revise a lista de produtos.
4. Clique em compartilhar catalogo.
5. O WhatsApp sera aberto com mensagem pronta.
6. Revise a mensagem.
7. Envie manualmente ao cliente.
8. Registre proxima acao para acompanhar retorno.

## Exemplos

- "Bom dia, segue nosso catalogo atualizado."
- "Posso te ajudar a escolher o melhor produto?"
- "Vou te chamar amanha para confirmar se ficou alguma duvida."

## Boas praticas

- Nao envie catalogo sem contexto.
- Personalize a mensagem quando souber a necessidade do cliente.
- Depois do envio, crie proxima acao.
- Use produtos ativos e atualizados.
- Acompanhe se o cliente respondeu.

## Erros comuns

- Enviar catalogo para lista fria sem acompanhamento.
- Achar que o sistema envia WhatsApp automaticamente.
- Compartilhar catalogo vazio.
- Nao registrar follow-up.

## Checklist

- [ ] Produtos ativos revisados.
- [ ] Catalogo aberto.
- [ ] Mensagem revisada no WhatsApp.
- [ ] Envio manual feito.
- [ ] Proxima acao criada.
