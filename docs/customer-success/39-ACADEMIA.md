# 39 - Academia

## Objetivo

Organizar trilhas de aprendizagem para cada perfil de usuario.

## Publico

Customer Success, treinamento, gestores, vendedores, administradores e suporte.

## Quando utilizar

Use para treinar novos clientes, novos funcionarios ou reciclar usuarios.

## Passo a passo

1. Escolha a trilha.
2. Leia os guias indicados.
3. Faça exercicio pratico.
4. Use checklist.
5. Valide aprendizado.

## Exemplos

- Trilha vendedor: Bem-vindo, Primeiro acesso, Guia do vendedor, Workspace, WhatsApp, Checklist diario.
- Trilha gestor: Bem-vindo, Guia do gestor, Dashboard, Indicadores, Health Score.
- Trilha admin: Primeiro acesso, Guia do administrador, Billing, Seguranca, Suporte.
- Trilha CS: Jornada, Implantacao, Sucesso, Retencao, Expansao.

## Boas praticas

- Treinar por perfil.
- Fazer exercicio com dado real.
- Reforcar depois de uma semana.
- Medir se houve uso.

## Erros comuns

- Dar a mesma aula para todos.
- Falar muito e praticar pouco.
- Nao acompanhar depois.
- Ignorar duvidas simples.

## Checklist

- [ ] Perfil escolhido.
- [ ] Guias lidos.
- [ ] Exercicio pratico feito.
- [ ] Duvidas respondidas.
- [ ] Aprendizado validado.
