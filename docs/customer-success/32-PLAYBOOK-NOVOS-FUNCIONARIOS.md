# 32 - Playbook novos funcionarios

## Objetivo

Treinar novas pessoas para usar o sistema sem depender de explicacao improvisada.

## Publico

Novos vendedores, novos gestores, administradores e responsaveis por treinamento.

## Quando utilizar

Use no primeiro dia de um novo funcionario ou quando alguem mudar de funcao.

## Passo a passo

1. Criar usuario individual.
2. Explicar empresa e papel.
3. Fazer primeiro login.
4. Mostrar dashboard.
5. Mostrar CRM.
6. Abrir Workspace.
7. Criar proxima acao de exemplo.
8. Enviar WhatsApp de teste quando adequado.
9. Entregar checklist diario.
10. Acompanhar por 3 dias.

## Exemplos

- Novo vendedor: foco em agenda, clientes, Workspace e WhatsApp.
- Novo gestor: foco em dashboard, recomendacoes e indicadores.
- Novo admin: foco em usuarios, plano e seguranca.

## Boas praticas

- Treinar com dados reais controlados.
- Dar uma tarefa simples ao final.
- Validar entendimento.
- Revisar acesso depois de uma semana.

## Erros comuns

- Entregar login sem treinamento.
- Usar conta de outro usuario.
- Ensinar tudo em uma vez.
- Nao acompanhar primeiros dias.

## Checklist

- [ ] Usuario criado.
- [ ] Papel correto.
- [ ] Login feito.
- [ ] Dashboard entendido.
- [ ] Workspace entendido.
- [ ] Checklist diario entregue.
