# Indice geral Customer Success

## Objetivo

Relacionar todos os documentos do programa de sucesso do cliente.

## Publico

Usuarios, gestores, suporte, CS, comercial, produto e treinamento.

## Quando utilizar

Use sempre que precisar encontrar o guia correto.

## Passo a passo

1. Identifique seu perfil.
2. Escolha o documento inicial.
3. Siga os documentos relacionados.
4. Use checklists para executar.

## Boas praticas

- Comece pelo perfil do usuario.
- Use guias curtos durante atendimento.
- Use playbooks para processos internos.
- Atualize o indice quando novo documento for criado.

## Erros comuns

- Enviar todos os documentos para um usuario novo.
- Usar playbook interno como manual do cliente.
- Esquecer de apontar o proximo passo.
