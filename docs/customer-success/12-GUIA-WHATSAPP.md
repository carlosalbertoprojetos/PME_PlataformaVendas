# 12 - Guia WhatsApp

## Objetivo

Ensinar quando e como usar WhatsApp na plataforma sem confundir link manual com envio automatico.

## Publico

Vendedores, gestores e suporte.

## Quando utilizar

Use para compartilhar catalogo, produto, pedido, mensagem para cliente ou follow-up.

## Passo a passo

1. Confira se o cliente tem telefone valido.
2. Clique no botao de WhatsApp.
3. O WhatsApp abre com mensagem pronta.
4. Leia a mensagem.
5. Ajuste o texto se necessario.
6. Envie manualmente.
7. Registre historico ou proxima acao.

## Exemplos

- Cliente sem recompra: "Oi, tudo bem? Vi que faz um tempo que nao compramos juntos."
- Pedido pendente: "Passando para confirmar se podemos seguir com o pedido."
- Produto recomendado: "Esse produto pode fazer sentido para sua proxima compra."

## Boas praticas

- Use mensagem pronta como ponto de partida.
- Personalize para parecer humano.
- Nao envie fora de horario inadequado.
- Respeite clientes que nao querem contato.
- Sempre acompanhe com proxima acao.

## Erros comuns

- Achar que o envio foi automatico.
- Enviar sem revisar telefone.
- Copiar mensagem sem personalizar.
- Nao registrar retorno.
- Usar WhatsApp para assunto sensivel sem contexto.

## Checklist

- [ ] Telefone valido.
- [ ] Mensagem revisada.
- [ ] Envio manual feito.
- [ ] Historico ou proxima acao registrado.
- [ ] Cliente nao foi abordado em excesso.
