# 38 - Certificacao

## Objetivo

Criar um programa simples de certificacao para garantir que usuarios saibam operar a plataforma.

## Publico

Vendedores, gestores, administradores, suporte e parceiros.

## Quando utilizar

Use apos treinamento inicial ou para validar novos funcionarios.

## Passo a passo

1. Concluir trilha basica.
2. Fazer atividade pratica.
3. Responder perguntas simples.
4. Demonstrar uso do Workspace.
5. Demonstrar criacao de proxima acao.
6. Receber aprovacao do gestor ou CS.

## Exemplos

- Certificacao vendedor: cliente, oportunidade, WhatsApp, proxima acao.
- Certificacao gestor: dashboard, recomendacoes, indicadores.
- Certificacao admin: usuarios, plano, seguranca.

## Boas praticas

- Avaliar pratica, nao decoracao.
- Usar caso real.
- Renovar quando produto mudar.
- Manter registro de aprovados.

## Erros comuns

- Certificar sem teste pratico.
- Criar prova longa.
- Ignorar perfil do usuario.
- Nao atualizar conteudo.

## Checklist

- [ ] Trilha concluida.
- [ ] Exercicio pratico feito.
- [ ] Perguntas respondidas.
- [ ] Uso demonstrado.
- [ ] Aprovacao registrada.
